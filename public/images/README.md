# Images Folder

## Required Image

Place your mascot image here as `mascot.png`

**File:** `mascot.png`
**Description:** A cartoon character (you!) in front of a rice cooker
**Recommended size:** 600x600px (transparent background PNG works best)
**Format:** PNG with transparent background preferred

This image will appear:
1. **On welcome screen** - Centered and large when you first open the app
2. **As floating buddy** - Small version in bottom-right corner while browsing recipes (bounces cutely!)

## How to Add Your Image

1. Create or find your cartoon mascot image
2. Save it as `mascot.png` in this folder
3. The app will automatically use it

That's it! No code changes needed.
